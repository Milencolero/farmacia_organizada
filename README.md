# Farmacia Organizada

Proyecto de gestión de farmacia con frontend y backend.

## Estructura del proyecto
- `front_farmacia/` → aplicación Vue.js
- `backend_farmacia/` → API Node.js / Express

## Instalación
1. Backend: `cd back && npm install`
2. Frontend: `cd front && npm install`

## Uso
- Ejecutar backend: `npm run dev`
- Ejecutar frontend: `npm run serve`
